BUG STORIES

Bug 调试故事

开发过程中的真实挑战与解决方案

01
主题切换崩溃 严重
切换深色/浅色主题时 GUI 偶发崩溃，排查发现是 egui 渲染状态未同步
+ 已修复: 同步 egui 渲染状态与主题配置
02
PLC 标签页空白 高
三菱/西门子 PLC 面板加载后内容为空，WebSocket 连接超时未重试
+ 已修复: 添加 WebSocket 重连与超时处理
03
CLI 启动参数丢失 高
Windows 下命令行参数被 PowerShell 截断，改用 clap 解析器修复
+ 已修复: 统一使用 clap 进行参数解析
04
连接保持异常 中
长时间空闲后串口连接断开，心跳包未正确处理 DTR/RTS 信号
+ 已修复: 实现心跳包与信号量管理
05
HEX 收发不一致 中
发送 HEX 0A 时被转换为换行符，需要区分原始字节与文本模式
+ 已修复: 严格区分二进制与文本传输模式
