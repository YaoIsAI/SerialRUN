CLI 命令示例

一行命令完成串口操作，告别 GUI 依赖

# 列出所有串口
$ serialrun list

# 交互式终端
$ serialrun interactive /dev/tty.usbmodem -b 115200

# 发送 AT 指令并读取响应
$ serialrun send-command COM3 "AT+CWMODE=1" --baud 115200

# HEX 模式发送 Modbus 帧
$ serialrun send /dev/ttyUSB0 "01 03 00 00 00 0A C5 CD" --hex

# 脚本回放
$ serialrun replay COM1 test_script.txt

Bug 调试故事

开发过程中遇到的真实挑战

严重

主题切换崩溃

切换深色/浅色主题时 GUI 偶发崩溃，排查发现是 egui 渲染状态未同步

高

PLC 标签页空白

三菱/西门子 PLC 面板加载后内容为空，根因是 WebSocket 连接超时未重试

高

CLI 启动参数丢失

Windows 下命令行参数被 PowerShell 截断，改用 clap 解析器修复

中

连接保持异常

长时间空闲后串口连接断开，心跳包机制未正确处理 DTR/RTS 信号

中

HEX 收发不一致

发送 HEX 0A 时被转换为换行符，需要区分原始字节与文本模式
