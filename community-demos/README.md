## 🤝 如何贡献

1. **Fork** 本仓库
2. 在 `community-demos/` 下创建子目录，命名格式：`demo-<场景英文名>-<你的名字>`
   - 示例：`demo-arduino-upload-yao`、`demo-plc-modbus-zhangsan`
3. 复制 `demo-template/` 下的所有文件作为起点
4. 按 [`demo-template/README.md`](./demo-template/README.md) 的格式要求填写所有内容
5. 提交 PR，标题格式：`[Community Demo] <场景> by <你的名字>`
6. 等待 Code Review（通常 1-3 个工作日）
7. 合并后会自动出现在上方 Demo 列表

## ✅ 审核标准

- 视频时长 ≤ 3 分钟，分辨率 ≥ 1080p
- 至少演示一个完整的 MCP 调试闭环（不是单纯打开软件）
- README 中英双语，关键操作配截图（≥ 5 张）
- 所有 prompt 文本可复用
- 引用 SerialRUN v0.4.3+ 正确仓库地址
- 无硬编码密钥、IP、个人路径等敏感信息
